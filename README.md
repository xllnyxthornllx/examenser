# Simulador de Examen - Servicios de Red

Este proyecto es una aplicación web interactiva diseñada para practicar y evaluar conocimientos sobre los servicios de red cubiertos en las primeras tres semanas del curso.

## Contenidos Evaluados
- **Semana 1: DNS** (Registros, comandos `dig`, peticiones recursivas).
- **Semana 2: Servicio Web** (HTTP/HTTPS, códigos de estado, métodos, APIs REST).
- **Semana 3: Servicio FTP** (Puertos 20/21, modos Activo/Pasivo, comandos básicos).

## Características
- **Feedback Inmediato:** Cada pregunta incluye una explicación técnica de la respuesta correcta.
- **Tipos de Preguntas:** Opción múltiple, Verdadero/Falso y completar comandos.
- **Diseño Responsivo:** Construido con Bootstrap 5 para funcionar en dispositivos móviles y escritorio.

## Cómo usar
Simplemente abre `index.html` en tu navegador. Completa las preguntas y haz clic en "Calificar Examen" para ver tu puntaje y las correcciones.
